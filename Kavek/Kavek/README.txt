Az adatbázisom egy kávézó adatait tartalmazza, különböző kávék fogyását egy adott időszakban. 
A program a következőképpen fog felépülni:
Egy oldalon megjelennek a kávék adatai, ahol különböző kritériumok szerint lehet majd törölni adatokat a listából.
Egy gombnyomás hatására törölni is fogja a kritériumnak megfelelőket. 
Végül kiírja majd csv-be a szűrt listát.
